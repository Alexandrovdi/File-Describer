# File Describer

Плагин для Obsidian, который автоматически создаёт и управляет заметками-описаниями для файлов в отслеживаемой папке. Идеально подходит для организации файловых архивов с поиском по описаниям, отслеживания изменений файлов и управления метаданными.

## Возможности

- **Автосоздание** — при добавлении файла в целевую папку автоматически создаётся заметка-описание
- **Модальное окно с вкладками** — две вкладки: «Новые файлы» (файлы без описания) и «Удалённые файлы» (удалённые файлы)
- **Отслеживание удаления** — при удалении файла заметка автоматически получает статус
- **Отслеживание переименования** — поле `File` в заметке обновляется при переименовании файла
- **Работа с осиротевшими заметками** — можно сохранить (отметить как просмотренное) или удалить заметку удалённого файла
- **Индикатор** — красный `!` на иконке панели при наличии ожидающих файлов
- **Рекурсивное сканирование** — отслеживание вложенных папок

## Как это работает

1. Поместите файл в целевую папку (по умолчанию: `Файлы хранения/`)
2. Плагин создаёт заметку-описание в `_заметки/` с frontmatter: `File Description`, `File` (викиссылка), `date_added`, `filetype`, `filename`
3. Нажмите на иконку плагина (или выполните команду), чтобы открыть окно просмотра
4. На вкладке «Новые файлы» добавьте описание и нажмите "✓" для сохранения
5. При удалении файла заметка автоматически получает статус `Status: "файл <name> - удален <date>"` — просмотрите во вкладке «Удалённые файлы»

## Установка

### Из браузера плагинов Obsidian

Найдите "File Describer" в Настройки → Сторонние плагины → Обзор.

### Через BRAT

1. Установите плагин [BRAT](https://github.com/TfTHacker/obsidian42-brat)
2. Добавьте `https://github.com/Alexandrovdi/File-Describer` в список бета-плагинов
3. Нажмите «Add Plugin»

### Вручную

1. Скачайте последний релиз с [GitHub Releases](https://github.com/Alexandrovdi/File-Describer/releases)
2. Распакуйте `main.js`, `manifest.json` и `styles.css` в `VaultFolder/.obsidian/plugins/file-describer/`
3. Включите плагин в Настройки → Сторонние плагины

## Настройки

| Настройка | Описание | По умолчанию |
|-----------|----------|--------------|
| Целевая папка | Папка для отслеживания файлов | `Файлы хранения/` |
| Папка заметок | Папка для заметок-описаний | `_заметки/` |
| Формат даты | Формат даты (Moment.js) | `YYYY-MM-DD` |
| Формат времени | Формат времени (Moment.js) | `HH:mm` |

## Разработка

```bash
git clone https://github.com/Alexandrovdi/File-Describer.git
cd File-Describer
npm install
npm run build    # сборка main.js
npm run dev      # режим слежения
```
